# flasklearn-secretsapce
学习flask的练手。计划写一个~~情侣空间~~ 在线论坛

## 安装
首先保证环境有python3,然后运行以下命令
`pip install -r requirements.txt`

## 运行
`flask run --reload`

## 预览

- 文章页面

  ![image-20211230214826522](README.assets/image-20211230214826522.png)


## 功能介绍

1. 帖子

    - 发帖
    - 编辑
    - 删除

2. 用户个人页面支持
    - 头像
    - 签名
    - 历史文章

3. 帖子回复
